= 2011年10月9日
中断了很久，重新继续移植。

= 2011年10月10日
实现基本地图界面。
可控制人物移动。
目前使用整体缩放方式，16号与12号字体显示效果不好。

= 2011年10月11日
完成：
	UI线程与游戏线程分离、
	图片使用缩放而文本使用匹配字号以保证最佳显示效果、
	创建人物界面、
	存档、取档、
	对话、战斗、学习、交易、任务系统等；
计划：
	增加触摸控制、
	修正一下逻辑BUG、
	微调体验、
	增加小游戏功能、
	增加设置功能……
	
新界面效果不错，:-)


= 2011年10月16日
周末状态不好，略改进了一些。

= 2011年10月30日
根据QQ334519667的建议，修改：
	NPC复活时，未恢复满血的BUG——这个是误用Java的数组复制，导致一个二维数组的副本其实根本就不是副本，所以NPC的状态没法回复，见NPC.InitData()。
	雪山派入门十三式不能用剑练功——属性数据用错了，见Player.PracticeSkill()中用Skill.skill_weapon_type[j1]来取真正的武器要求；
	加大石料任务的经验潜能涨幅——与悟性和福缘相关；
	调整了横屏时方向键位置；

= 2012年03月01日
修改战斗时吃药崩溃的BUG；
修改小游戏只能加一级的BUG。

= 2012年10月29日
添加存档的备份还原功能（基本原型），计划支持：
 区别联机档和娱乐档——
　　联机档有签名检验，玩家不可更改，以保证公平性；
　　娱乐档可以修改，以满足娱乐和测试；
　　联机档可以另存为娱乐档；
　　旧版本存档只能转为娱乐档；
　　联机只能在同类型的存档中进行。
 多个存档支持；
